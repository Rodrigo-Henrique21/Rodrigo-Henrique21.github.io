### 📋 Sobre o Projeto

Site portfólio desenvolvido para apresentar minha trajetória profissional, habilidades técnicas, projetos e certificações na área de **Engenharia de Dados** e **Arquitetura de Dados**. O site é totalmente estático, responsivo e otimizado para uma experiência fluida em qualquer dispositivo.

---

### 🖥️ Seções do Site

| Seção | Descrição |
|-------|-----------|
| **🏠 Hero** | Apresentação com foto, nome, cargo atual e links de contato |
| **👤 Sobre Mim** | Resumo profissional e destaques da carreira |
| **💼 Experiência** | Timeline interativa com +12 anos de experiência profissional |
| **🛠️ Projetos** | Repositórios em destaque do GitHub com linguagens e estatísticas |
| **⚡ Habilidades** | Grade visual de competências técnicas categorizadas |
| **📜 Certificações** | Certificações profissionais obtidas |
| **📬 Contato** | Formulário e links diretos para contato |

---

### 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e acessível
- **CSS3** — Tema escuro profissional, variáveis CSS, Flexbox, Grid e media queries
- **JavaScript (ES6+)** — Carregamento dinâmico de dados, animações com IntersectionObserver e interatividade
- **Font Awesome** — Ícones vetoriais
- **Google Fonts** — Tipografia (Inter)

### 📂 Estrutura do Projeto

```
portfolio-site/
```
